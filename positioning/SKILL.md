# Positioning Framework 

## What 
A complete positioning document that defines the product category, competition, persona overview, and unique differentiators with their challenges and values. 
Saved as: outputs/positioning.docx 

## How 

### Step 1 - Load context 
Read icp.md 
Read brand.md
Read voice.md
Read outputs/jtbd-analysis.docx 

### Step 2 - Write the product description
1 to 2 sentences covering: 
- Product name 
- Key features
- Problems being solved

Keep it concise and jargon free. 

### Step 3 - Write the category overview 
Outline the market the product competes in. 
Cover: 
- What category this product sits in 
- How this category is different from adjacent categories
- Any category level differentiators
- The emotional and situational state of buyers in this market 

### Step 4 - Map the competition 
Identify who buyers go to if they do not buy this product. 
For each competitor include:
- Competitor name
- Their core positioning
- Their key weakness compared to this product

Include direct competitors, indirect competitors, and the status quo option of doing nothing.

### Step 5 - Write the persona overview 
Describe the persona this positioning is built for. Cover:
- Title and seniority
- Core responsibilities
- What they care most about
- Their role in the buying process

If different personas have wildly different needs create a separate positioning document for each.

### Step 6 - Define unique differentiators
Identify 3 or more unique differentiators. A differentiator only qualifies if:
- It is unique to this product
- A competitor does not offer the exact same thing
- It directly addresses a customer challenge

For each differentiator define: 

Unique differentiator: What about this product makes it stand out from competition.
Challenge: What problem is the customer facing because this differentiator does not exist in their current solution. 
Value: How this differentiator solves that challenge. Draw on emotion, outcome vision, and specificity. 

### Step 7 - Review and align 
Check the full positioning document against:
- JTBD analysis output
- Competitor list
- ICP profile

Ensure the positioning:
- Is genuinely unique
- Addresses real customer challenges
- Connects features to outcomes
- Is free of generic claims 

### Step 8 - Produce output document 
Create a Word document with these sections in this exact order:

[Product name] Positioning for [Persona name] 

PRODUCT DESCRIPTION
1 to 2 sentences. 

CATEGORY OVERVIEW
3 to 5 sentences. 

THE COMPETITION
Create a table with three columns: Competitor name, Core positioning, Key weakness. One row per competitor. 

PERSONA OVERVIEW
3 to 5 sentences. 

UNIQUE DIFFERENTIATORS
Create a table with four columns: 
Label column, Differentiator 1, Differentiator 2, Differentiator 3. 
Row 1: Unique differentiator
Row 2: Challenge
Row 3: Value

Save as: outputs/positioning.docx
